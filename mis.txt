3.ბრჩხილები (მრგვალი (ფრჩხილები) აუცილებელია, რადგან print არის ფუნქცია და Python-ში ფუნქციის გამოძახება სწორედ ბრჩხილებით ხდება.)

4.ბრჩხილებს შიგნით ვწერთ იმას, რასაც ეკრანზე გვსურს რომ გამოვიდეს — მაგალითად ტექსტს, რიცხვს ან ცვლადს.

მაგალითი რიცხვით:

print(1)

მაგალითი ცვლადით:

name = "საბა"
print(name)


შეჯამება: print — არის ფუნქცია, რომელიც ინფორმაციას ბეჭდავს ეკრანზე.

მრგვალი ბრჩხილები საჭიროა, რადგან ფუნქცია სწორედ ასე გამოიძახება.

ბრჩხილებში ვწერთ იმას, რაც გვსურს გამოჩნდეს.

5.დეკლარაცია კოდინგში ნიშნავს, რომ პროგრამას ვეუბნებით ცვლადი ფუნქცია არსებობს
   
  int x; → დეკლარაცია
   x = 5; → ინიციალიზაცია
   int y = 10; → ორივე ერთად
 მოკლედ რომ ვთქვათ-დეკლარაცია ნიშნავს = გამოცხადება, ინიციალიზაცია = მნიშვნელობის მიცემა.
